# Code Plagiarism Detection System — Prototype

A full-screen web prototype of a **Code Submission & Plagiarism Detection System** for academic environments. Students submit programming assignments, instructors review similarity reports, and administrators manage the platform.

## Pages

| Page | Description |
|------|-------------|
| **Login** | Role-based sign-in (Student / Instructor / Admin) |
| **Student Dashboard** | View assignments, submission status, and timeline |
| **Instructor Dashboard** | Manage assignments, review submissions with similarity % |
| **Code Submission** | Drag-and-drop upload with format validation |
| **Similarity Report** | Side-by-side code comparison with match highlighting |
| **Admin Panel** | User management, security logs, system configuration |

## Presentation

A standalone **18-slide presentation** (`presentation.html`) covers the full system design — architecture, modules, requirements, decision tables, and tech stack. Navigate with arrow keys or on-screen buttons.

## Run

Open `index.html` in any browser. No build step required.

## Stack

HTML5, CSS3, JavaScript — zero dependencies.
